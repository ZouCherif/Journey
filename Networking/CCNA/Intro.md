# Introduction

- `Client` A device that access a service made available by a server.
- `Server` A device that provides services for clients.
- `Switches` Provides connectivity to hosts **_within the same LAN_**.
- `Routers` used to provide connectivity **_between LANs_**.
- `Firewalls` Control network traffic based on configured rules (placed inside or outside network), there is two types of firewalls: **_hardware_** firewalls and **_software/host based_** firewalls.
